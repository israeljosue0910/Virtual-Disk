IMPORTANTE: Si el programa termina sin desmontar o sin dar exit el disco en uso debe ser borrado ya que se
			torna defectuoso e inutil. Todos los metodos han sido probado, pero en caso de que se utilize uno y el
			programa falle, crear disco nuevo e intentar de nuevo, ya que como se menciono antes el disco es inutil 
			y el comando no va a funcionar como se supone.  

Para compilar y ejecutar el proyecto en eclipse se debe seguir los siguientes pasos:

	1. Buscar el proyecto en el lado izquierdo de la pantalla donde dice "Package
	   Explorer"
	2. Una vez localizado, expande el proyecto y expande el "source folder" src.
	   Luego expande el package "theSystem" localizado dentro de src.
	3. Dentro de "theSystem" van a haber tres clases. Presione el boton de la derecha del mouse sobre "MySystem", 
	   vaya a "Run As" y luego a "Java Application".
	4. Una ves haya sido compilado y ejecutado recuerde desmontar y dar exit al terminar de usar el programa
